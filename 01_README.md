# 📱 Megaline Mobile Plan Classification  
Machine learning and data analysis project focused on predicting whether telecom subscribers should switch to the **Smart** or **Ultra** mobile plan based on real usage behavior.

## 📌 Project Overview  
This project analyzes subscriber behavior data — including call duration, number of messages, and internet usage — to understand usage patterns and build a classification model that recommends the most suitable mobile plan.  
The goal was to achieve an accuracy of **0.75 or higher** on the test dataset.

This project demonstrates skills essential for **Data Analyst** and **Data Entry Specialist** roles, including data cleaning, validation, exploratory analysis, and model evaluation.

## 🧰 Tools & Technologies  
- **Python** (Pandas, NumPy, Scikit‑learn)  
- **Jupyter Notebook**  
- **Matplotlib / Seaborn**  
- **GitHub**  

## 📊 Key Responsibilities  
- Analyzed subscriber‑behavior data to understand feature distributions and usage patterns  
- Cleaned, validated, and prepared datasets for modeling  
- Split data into training, validation, and test sets for reliable evaluation  
- Compared multiple classification algorithms and tuned hyperparameters  
- Evaluated model accuracy against a 0.75 threshold  
- Performed interpretability checks to understand feature influence  

## 📈 Results  
- Achieved accuracy meeting the **0.75 requirement** on the test dataset  
- Identified key behavioral features that drive plan recommendations  
- Built a reliable model suitable for customer plan classification
